# Asset Notes

This starter includes simple SVG placeholder assets for:

- logo
- business icons
- payment icons
- dashboard icons
- a hero illustration placeholder

These files were created as starter placeholders so the repo stays lightweight and editable.

If you want to replace them later, good royalty-free sources include:

- [Heroicons](https://heroicons.com/)
- [Tabler Icons](https://tabler-icons.io/)
- [Phosphor Icons](https://phosphoricons.com/)
- [Pexels](https://www.pexels.com/)
- [Unsplash](https://unsplash.com/)
